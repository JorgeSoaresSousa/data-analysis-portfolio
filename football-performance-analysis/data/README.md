# Data

This folder contains all datasets used in the football performance analysis project.

## Structure

- **raw/**
  - Original, unmodified datasets as obtained from the source.
  - These files should not be altered.

- **processed/**
  - Cleaned and transformed datasets used for analysis.
  - These files are derived from the original data in the `raw` folder.

## Data Source

- All data used in this project is real.
- The datasets were collected from **FBref.com**.

## Notes

- Raw data was collected from FBref.com and stored in the `raw` folder.
- All data cleaning, transformation, and feature engineering were performed in Jupyter Notebooks using the raw data as input.
- The datasets in the `processed` folder represent the final, analysis-ready output of the notebooks.
- These processed datasets were used to build the Looker Studio dashboard and the final report.
